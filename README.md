## 项目组成：  
### 它将由两部分组成：
- 一个让人们查看和投票的公共站点。
- 一个让你能添加、修改和删除投票的管理站点。
### 模型：
&emsp;在这个简单的投票应用中，需要创建两个模型：问题 Question 和选项 Choice。
- Question 模型包括问题描述和发布时间。
- Choice 模型有两个字段，选项描述和当前得票数。每个选项属于一个问题。
